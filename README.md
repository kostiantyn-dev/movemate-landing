# MoveMate landing

Складніший демо-лендінг для портфоліо: преміальний сервіс переїздів для українського ринку.

## Стек

- HTML
- CSS
- JavaScript
- GitHub Pages-ready

## Функції

- повноекранний hero з локальним bitmap-зображенням;
- інтерактивний калькулятор вартості переїзду;
- перемикач сценаріїв переїзду;
- конструктор пакета послуг;
- FAQ accordion;
- мобільне меню;
- форма заявки з валідацією.

## Запуск

Відкрийте `index.html` у браузері або опублікуйте папку через GitHub Pages.
